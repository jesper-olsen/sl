# sl (Steam Locomotive)

Rust port of Masashi Toyoda's [sl](https://github.com/mtoyoda/sl) commandline utility.

## Getting Started

Install [Rust and Cargo](https://rustup.rs/). 

Clone the repository and run the game directly:

```bash
git clone https://github.com/jesper-olsen/sl.git
cd sl
cargo run
```
